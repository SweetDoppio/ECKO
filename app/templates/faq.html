FAQ - Frequently Asked Questions

What is EKHO?

EKHO is an educational platform designed to teach cybersecurity through interactive quizzes, scan simulations, and progress tracking. It’s built to make learning accessible and engaging for students and educators.

Is my data safe with EKHO?

Yes, EKHO operates in a safe, virtual environment. No real systems are affected during scans, ensuring you can learn without risking live networks.

How do I start using EKHO?

Simply sign up for an account on our website, log in, and begin with a scan or quiz. Educators can also create class accounts to monitor student progress.

What is a port?

A port is a virtual point on a computer where network connections start and end. It’s like a door number for data, with common ones like 80 (HTTP) and 443 (HTTPS) used for web traffic.

What is a firewall?

A firewall is a security tool that monitors and controls incoming and outgoing network traffic based on set rules, acting as a barrier between safe and unsafe networks.

What is malware?

Malware, short for malicious software, includes viruses, ransomware, and spyware designed to harm or exploit devices and data.

What is encryption?

Encryption is the process of converting data into a code to prevent unauthorized access, ensuring only those with the key can read it.
